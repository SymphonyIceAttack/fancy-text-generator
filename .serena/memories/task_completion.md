# Task Completion Checklist

## After Making Changes
1. Run `pnpm run lint` to check for linting issues
2. Run `pnpm run format` to format code
3. Test changes in browser (`pnpm dev` if not running)

## Before Committing
1. Verify all lint/format checks pass
2. Test functionality manually
3. Check for TypeScript errors in build: `pnpm run build`
